# Это проект связанное с сайтом школьной футбольной команды Адмирал ВМФ

Сам проект просто делает POST-запрос и отправляет форму со структурой MJML, переведенный на HTML на почту.
* * *
## Настройка

1. Нужно создать `.env` файл и заполнить в него такой структурой:
  ```env
  EMAIL_SENDER=<почта с которой будет отправляться письмо>
  EMAIL_RECEIVER=<почта на которую будет отправляться письмо>
  EMAIL_PASSWORD=<пароль от почты>
  ```
2. Затем нужно настроить почтовый сервер через `email_config.ini`:
```ini
[SMTP]
host = <Хост почтового сервера>
port = <Порт почтового сервера>
```
